1. Go to your telegram
2. Search @Botfather
3. Create a chatbot for get token api telegram

```
https://api.telegram.org/bot<YOUR_BOT_TOKEN>/getUpdates
```
5. Change
```
const telegramBotToken = 'your_token_bot';

```
